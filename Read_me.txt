def sort_phone_book
Функция для сортировки справочника. 
Копирует из файла данные в виде списка, сортирует список, перезаписывает файл.

def add_contact
Функция для добавления контакта.
На входе проверяет данные, на соответствие шаблону, за счет регулярных выражений.
Добавляет в файл данные нового контакта, запускает функцию сортировки.

def view_contacts_by_letter
Фнкция для вывода контактов с фамилиями, которые начинаются на заданную букву.
Заводит счетчик
Считывает построчно файл, выводит контакты с фамилиями, которые начинаются на назначенную букву.
Выводя - увеличивает счетчик.
Если счетчик остается пустым - говорит, что таких фамилий в справочнике нет.

def view_contacts
Функция выводит данные контактов постранично. Одна страница - одна буква, по алфавиту.
Считывает файл. Берет последовательно буквы алфавита и проходит по справочнику, сравнивая с 
первой буквой фамилии. После каждого прохода предлагает продолжить или прерваться.

def search_for_contacts
Функция для поиска контакта по одному или нескольким вводным.
Заводит счетчик.
Запрашивает по каким данным производить поиск, далее сравнивает фамилию с фамилией, имя с именем и т.д.
Проходясь построчно по контактам в файле.
Если счетчик пуст - говорит, что контакта с такими данными нет в справочнике.

def changing_a_contact
Функция для изменения данных контакта.
Запрашивает фамилию, имя и отчество искомого контакта. Заводит счетчик. 
Проходясь построчно по файлу, сравнивает вводные данные с даннми контактов, в случае совпадения 
предлагает поочередно заменить данные или оставить прежними, т.е. можно поменять только название фирмы, 
остальное она возьмет из старой записи. Обновленный контакт она сохраняет в новый список, 
также как и контакты не подошедшие под запрос. В конце прохода, если счетчик не пуст, перезаписывает 
файл из обновленного списка контактов.
